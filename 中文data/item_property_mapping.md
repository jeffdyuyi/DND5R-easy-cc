# 5etools 物品属性代码映射 (Item Property Mapping)

整理 D&D 5e 工具集中的物品/武器属性代码与其对应的中文描述。

| 代码 | 中文名称 | 英文名称 | 说明 |
| :--- | :--- | :--- | :--- |
| **武器属性** | | | |
| V | 多用 | Versatile | 可以用单手或双手使用，伤害不同。 |
| F | 灵巧 | Finesse | 可以选择使用力量或敏捷进行攻击。 |
| H | 重型 | Heavy | 对小型种族有劣势。 |
| R | 触及 | Reach | 攻击范围增加 5 尺。 |
| T | 投掷 | Thrown | 可以作为远程武器投掷。 |
| L | 轻型 | Light | 适合双武器战斗。 |
| 2H | 双手 | Two-Handed | 必须用双手使用。 |
| A | 弹药 | Ammunition | 需要消耗弹药。 |
| LD | 装填 | Loading | 每个动作/附赠动作/反应只能射击一次。 |
| **物品标准属性** | | | |
| CNS | 消耗品 | Consumable | 使用后会损耗或消失。 |
| ST | 法杖 | Staff | 魔法法杖属性。 |
| RD | 权杖 | Rod | 魔法权杖属性。 |
| WD | 魔杖 | Wand | 魔法魔杖属性。 |
| SCF | 施法法器 | Spellcasting Focus | 作为法表的法器使用。 |
| **特殊属性** | | | |
| AF | 火器 | Firearm | 现代或特殊火器。 |
| BF | 连发 | Burst Fire | 可以进行扫射。 |

> [!TIP]
> 自动化脚本在处理武器数据时，可以根据这些映射生成更友好的 UI 展示文字。
