# 中文数据仓库 (Database Resource Memo)

这个文件夹用于存放提取和整合后的 D&D 5e 资源数据，仅作为数据库资源备忘使用。

## 注意事项
- **非编程资源**：本文件夹及其子内容不参与项目的任何编程逻辑或代码引用。
- **纯净分类**：旨在提供一个清晰、纯净的数据视图，方便手动查阅和整合。
