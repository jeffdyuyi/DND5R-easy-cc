# 第三方自制内容整合指南 (Homebrew Integration)

由于网络环境限制，部分远程自制内容（如 《歪曲之月》TCM、《格里姆霍洛》GH 等）无法自动完成全量下载。为了保证数据的完整性，建议你手动下载相关 JSON 文件并放入本目录下。

## 远程仓库地址
- **主仓库 (tjliqy)**: `https://github.com/tjliqy/homebrew/tree/cn`
- **加速下载代理**: `https://gh-proxy.org/`

## 建议下载的文件清单
你可以直接通过浏览器访问以下链接并将其内容保存为 `.json` 文件归档到本目录下：

1. **歪曲之月 (The Crooked Moon - TCM)**:
   - [class-bard-tcm.json](https://gh-proxy.org/https://raw.githubusercontent.com/tjliqy/homebrew/cn/data/class/class-bard.json) (注：远程库中部分职业已合并)
2. **格里姆霍洛 (Grim Hollow - GH)**:
   - [class-fighter-gh.json](https://gh-proxy.org/https://raw.githubusercontent.com/tjliqy/homebrew/cn/data/class/class-fighter.json)

## 操作建议
- 下载后的文件可以直接放入 `20_第三方自制内容/01_自制职业与子职业/`。
- 如果你需要我提供像 `01_职业与子职业` 那样的“一键拆分”脚本来处理这些自制文件，只需将下载好的文件丢进本地目录后告知我即可。

*注：本地已完整收录 FRHoF (皓月学院等) 及 XPHB (2024版) 内容，无需额外下载。*
