## 二、子职业特性更新

### 3级特性

#### 1. 战斗激励 (Combat Inspiration) ✅

**更新要点：**
- 格式优化：使用标题分隔防御和进攻两种效应
- 文本精确化："巧言"代替"巧辩"
- 添加英文对照：Defense、Offense

**效应说明：**
- **防御**：反应使用，将骰值加到AC，可能使攻击失手
- **进攻**：攻击命中后立即使用，将骰值加到伤害

#### 2. 战争训练 (Martial Training) ✅

**重要更新：**
```typescript
grants: {
    weaponProficiencies: ["军用武器"],
    armorProficiencies: ["中甲", "盾牌"]
}
```

**新增功能：**
- 添加 `grants` 字段，系统化地声明授予的熟练项
- 确保这些熟练项能在角色卡中正确显示
- 支持自动应用到角色数据

**特性内容：**
- 获得**军用武器熟练**
- 获得**中甲和盾牌**的护甲受训
- 可使用简易或军用武器作为吟游诗人法术的施法法器

## 三、类型系统增强

### ClassFeature 接口更新

为支持勇气学院的战争训练特性授予熟练项，更新了 `ClassFeature` 接口：

```typescript
export interface ClassFeature {
  name: string;
  description: string;
  level: number;
  grants?: {
    weaponProficiencies?: string[];
    armorProficiencies?: string[];
    skillProficiencies?: string[];
    toolProficiencies?: string[];
    savingThrows?: string[];
  };
}
```

**支持的熟练项类型：**
1. **武器熟练** (weaponProficiencies)
2. **护甲熟练** (armorProficiencies)
3. **技能熟练** (skillProficiencies)
4. **工具熟练** (toolProficiencies)
5. **豁免熟练** (savingThrows)

### 1. 熟练项自动应用

**需要实现的逻辑：**
```typescript
// 伪代码
function applySubclassFeatures(character, subclass, level) {
  const features = subclass.features.filter(f => f.level <= level);
  
  features.forEach(feature => {
    if (feature.grants) {
      // 应用武器熟练
      if (feature.grants.weaponProficiencies) {
        character.proficiencies.weapons.push(
          ...feature.grants.weaponProficiencies
        );
      }
      
      // 应用护甲熟练
      if (feature.grants.armorProficiencies) {
        character.proficiencies.armor.push(
          ...feature.grants.armorProficiencies
        );
      }
      
      // 其他熟练项同理...
    }
  });
}
```

## 六、勇气学院吟游诗人概览

### 核心定位
**战斗型吟游诗人** - 能够在前线战斗并支持队友

### 关键特性

**3级：**
- 🛡️ **战斗激励** - 增强诗人激励的战术价值
- ⚔️ **战争训练** - 获得军用武器、中甲、盾牌熟练

**6级：**
- ⚡ **额外攻击** - 每回合两次攻击，可用戏法替换

**14级：**
- 🔮 **战斗魔法** - 施法后可附赠动作攻击

### 装备建议

**武器选择：**
- 长剑（军用，d8）
- 战斧（军用，d8）
- 长弓（军用，d8）

**护甲选择：**
- 链甲衫（中甲，AC 13 + 敏捷调整值最高2）
- 胸甲（中甲，AC 14 + 敏捷调整值最高2）
- 盾牌（+2 AC）

**典型装备：**
- 胸甲 + 盾牌 + 长剑 = AC 18（14+2+2, 假设敏捷调整值≥2）
- 比基础吟游诗人的轻甲高得多

### 战术定位

1. **前线支援** - 可以站在前排保护队友
2. **多功能输出** - 近战攻击 + 法术施展
3. **激励增强** - 诗人激励可用于防御和进攻
