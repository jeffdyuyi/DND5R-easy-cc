## 二、子职业特性更新

### 3级特性

#### 炫目舞步 (Dazzling Footwork) ✅

这是舞蹈学院的核心特性，包含**四个子效应**：

**格式优化：**
- 统一使用段落分隔（`\n\n`）提高可读性
- 每个子效应添加英文对照
- 移除项目符号（`•`），使用**标题格式**

**1. 大舞蹈家 Dance Virtuoso**
- 舞蹈相关的魅力（表演）检定具有**优势**
- 适用场景：表演、娱乐观众、吸引注意

**2. 无甲防御 Unarmored Defense**
```
AC = 10 + 敏捷调整值 + 魅力调整值
```
- 与武僧的无甲防御类似，但使用魅力而非智慧
- 高属性时AC可达：10 + 5(敏捷) + 5(魅力) = **20 AC**！
- 必须**未着装护甲且未持用盾牌**

**3. 灵巧打击 Agile Strikes**
- 当消耗诗人激励使用次数时，可附带一次**徒手打击**
- 作为同一动作/附赠动作/反应的一部分
- 提供了额外输出机会

**4. 诗人痛击 Bardic Damage**
- 徒手打击使用**敏捷**代替力量（适合高敏吟游诗人）
- 伤害 = **诗人激励骰 + 敏捷调整值**
- **不消耗诗人激励使用次数**（key！）
- 伤害类型：钝击

**伤害计算示例：**
```
5级舞蹈学院吟游诗人（敏捷+3）
诗人骰：d8
徒手打击伤害：d8 + 3 = 平均 7.5伤害
（比普通徒手打击的1+3=4伤害高得多）

15级时（敏捷+5）
诗人骰：d12
徒手打击伤害：d12 + 5 = 平均 11.5伤害
```

**用词更新：**
- "诗人伤害" → "诗人痛击"（更贴合战斗语境）
- "以一次" → "以一次"（保持准确）

### 6级特性（两个）

#### 1. 鼓舞之移 (Inspiring Movement) ✅

**更新要点：**
- 统一用词："能够以反应"代替"可以执行反应"
- 简化表达："他的反应"代替"他自己的反应"
- 移除不必要的空格

**战术价值：**
- **反应移动**：敌人结束回合在5尺内触发
- **团队移动**：盟友也可移动（30尺内）
- **不引发借机**：安全撤退或重新定位
- **消耗**：1次诗人激励使用次数

**使用场景：**
```
敌人近战单位移动到你身边攻击并结束回合
→ 你使用反应移动15尺（速度30÷2）
→ 盟友也可移动（如射手远离危险）
→ 敌人下回合需要再次移动接近（浪费行动）
```

#### 2. 协同舞步 (Tandem Footwork) ✅

**更新要点：**
- 用词统一："未陷入失能状态"代替"不处于失能状态"
- 简化指代："该骰值"代替"这次诗人骰骰值"
- 移除不必要空格

**先攻增益：**
- 影响范围：自己 + 30尺内所有盟友
- 加值：= 诗人激励骰的掷骰结果
- 条件：能听见或看见你
- 消耗：1次诗人激励使用次数

**价值分析：**
```
5级吟游诗人（诗人骰d8）
平均加值：+4.5 先攻
影响4名队友 = 总增益约+18先攻优势

10级（诗人骰d10）
平均加值：+5.5 先攻

关键战斗可确保团队先手！
```

### 14级特性

#### 引导闪避 (Leading Evasion) ✅

**更新要点：**
- 调整叙述顺序："若你陷入失能状态"放在最后
- 用词统一："陷入"代替"处于"

**效果说明：**

**个人效果（类似盗贼闪避）：**
- 豁免**成功**：不受伤害
- 豁免**失败**：只受一半伤害

**团队效果（独特）：**
- 5尺内的盟友也获得此增益
- 非常适合保护队友

**限制：**
- 失能状态时无法使用

**实战价值：**
```
火球术（8d6伤害，DC 15敏捷豁免）

普通角色：
- 豁免成功：14伤害
- 豁免失败：28伤害

舞蹈学院吟游诗人：
- 豁免成功：0伤害 ✨
- 豁免失败：14伤害

且能保护身边的盟友！
```

## 三、舞蹈学院吟游诗人概览

### 核心定位
**高机动性无甲战斗吟游诗人** - 优雅的战场舞者

### 关键数据

**AC计算（无甲防御）：**
```
1级（敏捷14，魅力16）：10 + 2 + 3 = 15 AC
4级（敏捷16，魅力18）：10 + 3 + 4 = 17 AC
8级（敏捷18，魅力20）：10 + 4 + 5 = 19 AC
12级+（敏捷20，魅力20）：10 + 5 + 5 = 20 AC
```
与重甲相当，且无需护甲！

**徒手打击伤害：**
```
等级  诗人骰  敏捷  平均伤害
1-4   d6     +3    6.5
5-9   d8     +4    8.5
10-14 d10    +5    10.5
15-20 d12    +5    11.5
```

### 战术风格

**1. 高机动战场控制**
- 反应移动（鼓舞之移）
- 不引发借机攻击
- 带动盟友重新定位

**2. 先攻优势**
- 协同舞步提升全队先攻
- 确保己方先手权

**3. 生存能力强**
- 高AC（无甲防御）
- 类似盗贼的闪避（引导闪避）
- 保护周围盟友

**4. 近战辅助输出**
- 诗人痛击提供可观的徒手伤害
- 使用诗人激励时附带攻击（灵巧打击）

### 属性优先级

1. **魅力**（主属性）- 施法DC、AC、技能
2. **敏捷**（次要）- AC、攻击、先攻
3. **体质**（生存）- 生命值

理想属性分配（标准数组）：
- 力量：8
- 敏捷：15 (+1种族) = 16
- 体质：13 (+1种族) = 14
- 智力：10
- 感知：12
- 魅力：14 (+2种族) = 16

### 装备建议

**武器：**
- 主要：徒手打击（诗人痛击）
- 备用：细剑（优雅武器，使用敏捷）

**护甲：**
- **无**（使用无甲防御）
- 时尚的舞者服装✨

**配件：**
- 乐器（施法法器）
- 舞者的饰品

## 七、系统集成建议

### 1. AC计算系统

需要在角色卡中实现无甲防御计算：

```typescript
function calculateAC(character) {
  // 检查是否为舞蹈学院吟游诗人
  if (character.subclass === "bard-dance" && character.level >= 3) {
    // 检查是否未穿护甲且未持盾牌
    if (!character.armor && !character.shield) {
      const unarmoredAC = 10 + character.dexMod + character.chaMod;
      // 与其他AC来源比较，取最高值
      return Math.max(character.baseAC, unarmoredAC);
    }
  }
  return character.baseAC;
}
```

### 2. 诗人痛击伤害

在徒手打击伤害计算中添加特殊处理：

```typescript
function calculateUnarmedDamage(character) {
  if (character.subclass === "bard-dance" && character.level >= 3) {
    const bardicDie = getBardicDie(character.level);
    const damage = `${bardicDie} + ${character.dexMod}`;
    return {
      formula: damage,
      type: "钝击",
      note: "诗人痛击"
    };
  }
  return standardUnarmedDamage;
}
```
